Building Examples

Most examples can be build with either Gradle or Maven unless otherwise stated. Make sure to look for readme.html and readme.txt for further instruction.

Maven Build Example

~$ mvn -pl org.icepdf.os.examples:png-capture -am package

Gradle Build Example

~$ gradle :examples:capture:png:assemble
