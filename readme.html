Building the ICEpdf library from source

The ICEpdf distribution bundle contains libs-sources folder that contains the source for the core and viewer modules. This was intended for incorporation with a developers IDE and was not designed for building the library from source.

Building the ICEpdf library from source first requires that the libraries source is checked from github https://github.com/pcorless/icepdf. Once downloaded the project can be build using Gradle or Maven. Similarly, the examples can only be build from the svn checkout. Code references for examples can also be found on the wiki.
